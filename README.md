# House Hunting Landing Page - React + Vite
A simple landing page built using React and Vite for an online House Hunting platform.

# Features
- Responsive layout that adjusts to all screen sizes
- House listings categorized by location (Eldoret, Nairobi, Mombasa, Kisumu, Kiambu)
- Modern UI with a brown primary theme
- Fast performance powered by Vite and React components

# Requirements
- Node.js and npm installed
- React (v18+)
- Vite bundler
- Code editor such as VS Code
- Git for version control

# Setup Instructions
#Clone the repository
git clone https://github.com/your-username/your-repo-name.git

#Move into the project directory
cd your-repo-name

#Install project dependencies
npm install

#Start the development server
npm run dev

# Endpoints
Since this is a landing page, there are no backend endpoints.
However, the available frontend routes (if using React Router) could be:
/ → Landing Page
/locations → List of house locations
/contact → Contact / Inquiry section

# Common Issues and Solutions
- Error: npm: command not found
Solution: Install Node.js from the official website (includes npm).
- Error: Failed to resolve module …
Solution: Run npm install to reinstall dependencies.
- Error: Page not styling correctly
Solution: Ensure CSS files are imported properly and restart the dev server.
