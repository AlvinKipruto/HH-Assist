# House Hunting Landing Page - React + Vite
A simple landing page built using React and Vite for an online House Hunting platform.

# Features
-Responsive layout that adjusts to all screen sizes
- House listings categorized by location (Eldoret, Nairobi, Mombasa, Kisumu, Kiambu)
- Modern UI with a brown primary theme
- Fast performance powered by Vite and React components
